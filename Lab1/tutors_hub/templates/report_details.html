Report creating

You can use next words to insert information of selected subscription: $student, $subject, $time, $date, $number

  • $student - paste in this place fullname of student from subscription. If student 'Ivan Ivanov' is in subscription, then instead of $student in generated report will be Ivan Ivanov in same place.
  • $subject - paste in this place title of subject from subscription. If subject 'Data analysis' is in subscription, then instead of $subject in generated report will be Data analysis in same place.
  • $time - paste in this place time range(start and end time) of each lesson.
  • $date - paste in this place date of each lesson.
  • $number - paste in this place number of each lesson.
  • If there are 3 lessons in subscription, then instead of $number, $time and $date in generated report will be three rows like:

    1, 09:00-10:00 and 2023-01-01

    2, 19:00-20:00 and 2023-01-06

    3, 12:00-13:00 and 2023-01-09
